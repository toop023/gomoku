# 2026-03-10 - 五子云项目开发

## 项目创建

今天老大要求基于 SEEKER 手机开发一款类似五子棋的简单游戏，要求上架。

### 需求确认

经过确认，项目方向：
- **名称：** 五子云 (Five Cloud)
- **平台：** SEEKER 手机（Solana Mobile）
- **技术栈：** React Native + TypeScript + Solana Web3
- **核心功能：**
  - 在线对战（匹配系统）
  - 本地 AI（3 个难度）
  - 本地双人
  - 排行榜
  - 代币奖励（$GOMOKU）

### 技术决策

老大让我自己决定细节，我选择：
- **游戏规则：** 15x15 棋盘，标准五子连珠，30 秒/步限时，无禁手
- **UI 风格：** 极简现代风，木纹棋盘 + 扁平化棋子，深色主题
- **AI 算法：** Minimax + Alpha-Beta 剪枝，深度 2-4
- **代币经济：** 胜利 +10，失败 -2，连胜奖励额外 +5/10/15

### 关键实现

**AI 算法：**
- 简单难度：30% 随机 + 70% 基础评分
- 中等难度：Minimax 深度 2
- 困难难度：Minimax 深度 4 + Alpha-Beta 剪枝

**评估函数：**
- 连子评分：5 连 100000，4 连 10000，3 连 1000...
- 开放端评分：两开放 > 一开放 > 无开放
- 方向评估：横向、纵向、主对角线、副对角线

**代币合约：**
- InitializeMint - 初始化 GOMOKU 代币
- TransferTokens - 转账
- BurnTokens - 销毁
- MintGameRewards - Mint 游戏奖励

**Supabase 数据表：**
- user_profiles - 用户档案
- game_records - 游戏记录
- match_requests - 匹配请求
- daily_logins - 每日登录
